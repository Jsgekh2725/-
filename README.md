# DirectDrop

موقع رفع ملفات احترافي يقبل الاستضافة على GitHub Pages.

## المميزات
- تصميم احترافي جداً
- Glassmorphism UI
- أنميشن للأزرار والخلفية
- شاشة تحميل متحركة
- إنشاء رابط مباشر للتحميل
- متوافق مع الجوال
- جاهز للاستضافة على GitHub Pages

## طريقة التشغيل
افتح ملف:
index.html

## رفعه على GitHub Pages
1- ارفع الملفات إلى Repository جديد  
2- ادخل Settings  
3- Pages  
4- اختر Branch = main  
5- Save  

سيعمل الموقع مباشرة.
